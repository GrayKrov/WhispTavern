# WhispTavern Website

> **Private Repository** – For internal development and collaboration only.

Welcome to the **WhispTavern** website project — the official community hub for adventurers, storytellers, and creative worldbuilders.

---

## 🎯 Project Overview & Goals

WhispTavern is designed to be a clean, welcoming, and responsive website that serves as the gateway to our community, providing easy access to resources, announcements, and social channels.

### Goals for Key Pages

| Page                | Purpose & Goals                                       |
|---------------------|-----------------------------------------------------|
| **Home (`index.html`)**       | Serve as the main landing page, introduce WhispTavern, showcase announcements, and invite users to join Discord. |
| **About Us (`Pages/About.html`)** | Share the community’s story, mission, and values to welcome newcomers and build trust. |
| **Community Resources (`Pages/Resources.html`)** | Provide curated tools, guides, and links to help members with worldbuilding, storytelling, and gameplay. |
| **Discord Link**               | Connect visitors directly to our active Discord community for real-time chat and events. |

---

## 🛠 Technologies & Tools Used

- **HTML5** – Semantic, accessible markup  
- **CSS3** – Styling with CSS Grid and Flexbox for responsive layouts  
- **JavaScript (vanilla)** – Basic interactivity (located in `Scripts/Main.js`)  
- **Git & GitHub** – Version control and private repository hosting  
- **Visual Studio Code** – Primary code editor
